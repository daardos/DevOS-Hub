# 🚀 DevOS Hub Elite // v7.5 Ultimate Suite

**DevOS Hub** — это мощная, стильная и полностью автономная веб-среда (IDE Sandbox), созданная специально для разработчиков. Это «швейцарский нож» в твоем браузере, объединяющий написание кода, планирование спринтов и инструменты обработки данных в одном футуристичном интерфейсе.

---

## 🛠 Ключевые возможности

### 💻 IDE Sandbox (Elite Edition)
*   **Движок Ace Editor**: Полноценная подсветка синтаксиса и интеллектуальное автодополнение (IntelliSense) при работе через сервер/GitHub.
*   **Instant Execution**: Тестируй JS-алгоритмы прямо в браузере с выводом в стилизованный терминал.
*   **Smart Fallback**: Автоматическое переключение в Offline-режим (стандартный текстовый редактор), если интернет недоступен.

### 📋 Sprint Planner & Library
*   **Task Manager**: Удобный трекер задач с сохранением состояния в `localStorage` (данные не пропадут после перезагрузки).
*   **Snippet Library**: Персональное хранилище кода с функциями поиска, добавления и быстрого удаления.

### 🔧 DevToolbox & API Lab
*   **JSON Formatter**: Быстрая валидация и «прихорашивание» сложных JSON-структур.
*   **Password Generator**: Генерация криптографически стойких паролей с настройкой длины.
*   **API Checker**: Полноценная лаборатория для GET-запросов (поддержка JSON и HTML-страниц).
*   **Base64 & Lorem Ipsum**: Быстрые утилиты для повседневной рутины кодера.

## 🎨 Визуальная эстетика
*   **Dark Engineering Theme**: Глубокие темно-синие градиенты с наложением инженерной сетки.
*   **Glassmorphism UI**: Эффект матового стекла, плавные анимации и неоновое свечение элементов.
*   **⚡ CORE FX**: Уникальный анимированный фон в виде математической нейросети на Canvas (включается секретной кнопкой в сайдбаре).

## 🚀 Быстрый старт

Проект полностью автономен и состоит из одного файла.

1.  **Скачайте** файл `index.html`.
2.  **Запустите** его в любом современном браузере.
3.  *Для работы автодополнения рекомендуется использовать GitHub Pages или любой локальный сервер.*

## 🧰 Стек технологий
*   **Frontend**: HTML5, CSS3 (Flexbox, Grid, Glassmorphism).

*   **Editor**: Ace Editor API (Integrated via CDN).
